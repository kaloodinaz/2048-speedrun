2048 SpeedRun is an extended version of the classic 2048 game, featuring a 4x4 grid instead of the standard 4x4 board. This variation allows for a more strategic approach to merging tiles, making it a great option for players who enjoy longer gameplay sessions.

This SpeedRun fast paced version by Kaloo Dinaz was cloned from the "non-official" 2048 game authored by Kaloo Dinaz.
